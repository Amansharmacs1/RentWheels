# 🚗 RentWheels

RentWheels is a full-stack MERN web application that connects vehicle owners with customers looking to rent cars and bikes. Owners can list their vehicles, while customers can discover and book them through a simple and user-friendly interface with an offline payment workflow.

> **Project Status:** 🚧 Currently in Development (Phase 1 Completed)

---

## ✨ Features

### Phase 1

* User Registration & Login
* JWT Authentication
* Role-Based Access (Customer & Owner)
* Protected Routes
* User Profile Management
* Responsive Landing Page
* Welcome Email using EmailJS
* Secure Password Hashing with bcrypt

### Upcoming Features

* Vehicle Listing & Management
* Vehicle Search & Filters
* Booking System
* Offline Payment Confirmation
* Booking History
* Reviews & Ratings
* Owner Dashboard
* Admin Dashboard
* Email Notifications
* Analytics & Reports
